# The [A]udio [V]ideo and [L]ive Player

### Installation

```shell
npm install
```

### Start Dev server

```shell
npm start
```

### Create a Build

```shell
npm run build
```

### Run Tests

```shell
npm run test
```

### Create Docs

```shell
npm run doc
```
